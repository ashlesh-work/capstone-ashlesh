# Who Title III covers

ADA Title III applies to private businesses that are open to the public ("places of
public accommodation").

# The standard situation

DOJ has not issued a Title III regulation that prescribes a single mandatory web
technical standard. However, DOJ guidance states that inaccessible web features can
deny people with disabilities equal access to a business's goods and services, and it
points organizations to WCAG and the Section 508 Standards as useful technical
references.

# Practical target

Because DOJ identifies the same barrier categories that WCAG addresses — contrast,
text alternatives, captions, labels, keyboard access, zoom, headings, and error
messaging — building materially below WCAG is difficult to defend. A practical
governance target is WCAG 2.1 AA at minimum and WCAG 2.2 AA preferred.

# Note on legal interpretation

Title III exposure is fact-specific. This page is a general summary; counsel should
assess a specific business's obligations.
