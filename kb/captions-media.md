# Captions for prerecorded video

Prerecorded video that contains audio must have synchronized captions (SC 1.2.2).
This covers onboarding videos, product tours, and explainer clips.

# Audio description or media alternative

When prerecorded video conveys meaningful information visually that is not spoken in
the narration, provide an audio description or a full text alternative (SC 1.2.3 and
SC 1.2.5).

# Live captions

If the site streams live events — webinars, support sessions, or live assistant or
avatar events presented as synchronized media — those need live captions (SC 1.2.4).

# Transcripts for conversational audio

For a pure conversational voice assistant that is not synchronized media, a
persistent, real-time text transcript of both sides of the conversation is the most
effective accessibility equivalent. Render every assistant utterance and every
recognized user utterance as visible text.
